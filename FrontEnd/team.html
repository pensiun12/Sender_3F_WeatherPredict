Tentang Kami

1. Profil Tim

Kami adalah tim mahasiswa dari program studi Jaringan Telekomunikasi Digital yang membentuk tim bernama Sender_3F. Proyek Weather Predict ini dikembangkan sebagai bagian dari tugas akhir dalam mata kuliah Desain Proyek.

2. Tujuan Proyek

Tujuan utama dari proyek ini adalah untuk menyediakan layanan prakiraan cuaca secara real-time dan ramah pengguna, dengan antarmuka web serta fitur chatbot untuk menjawab pertanyaan terkait cuaca secara otomatis.

3. Teknologi yang Digunakan

  • Frontend: HTML, CSS, JavaScript
  • Backend: PHP (Laravel)
  • AI: TensorFlow (untuk chatbot)
  • Database: MySQL
  • Desain UI/UX: Figma
  • API Cuaca: OpenWeatherMap (https://openweathermap.org/)
